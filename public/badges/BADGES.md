# Badge Folder Documentation

All badge files must be placed in `/public/badges/` directory 
Thumbnail versions go in `/public/badges/thumb/` as `.webp` files.
File versions go in `/public/badges/` as `.gif` files.

## Required Badge GIF Files

| Badge Name | GIF Filename | Description |
|------------|--------------|-------------|
| First Pixel | `firstpixel.gif` | Awarded when user places their first pixel |
| Pixel Master | `pixelmaster.gif` | Awarded for placing 10,000 pixels in a day (Daily Objective) |
| Weekly Warrior | `weeklywarrior.gif` | Awarded for placing 50,000 pixels in a week (Weekly Objective) |
| Chat Greeter | `chatgreeter.gif` | Awarded for saying hello in chat (Daily Objective) |
| Faction Leader | `factionleader.gif` | Awarded to faction owners |
| TOTW Winner | `totwwinner.gif` | Awarded to Team of the Week winners |
| TOTW Most Improved | `totwmostimproved.gif` | Awarded for Most Improved in TOTW |
| TOTW Underdog | `totwunderdog.gif` | Awarded for Underdog award in TOTW |
| TOTW Community Choice | `totwcommunitychoice.gif` | Awarded for Community Choice in TOTW |
| Veteran | `veteran.gif` | Awarded for 1 year of account age |
| Premium | `premium.gif` | Awarded to premium members |
| VIP | `vip.gif` | Awarded to VIP members |
| Bug Hunter | `bughunter.gif` | Awarded for reporting bugs |
| Template Master | `templatemaster.gif` | Awarded for completing 10 templates |
| Social Butterfly | `socialbutterfly.gif` | Awarded for 1000 chat messages |

## Thumbnail Requirements

- Format: WebP
- Size: 48x48 pixels recommended
- Location: `/public/badges/thumb/{badgename}.webp`

## Full Badge Requirements

- Format: GIF (animated) or WebP
- Size: 128x128 pixels recommended
- Location: `/public/badges/{badgename}.gif`
